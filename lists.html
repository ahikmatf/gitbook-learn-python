Array

Array

Array adalah sekumpulan data dalam suatu object yang memiliki data type sama atau berbeda

myList = [1, "Hello", 3, "Word", 5]
print(myList[0]) #output 1
print(myList[1]) #output Hello

myList.append("Makers") # untuk menambah isi array
myList.pop(2) # Remove array di index ke 2
myList.insert(0, 300) # Insert nilai 300 di index array ke 0
myList.index("Hello") # Mencari index yg berisi string "Hello"
myList.sort() # Sorting dari value terkecil keterbesar
myList.reverse() # Sorting value dari index terbesar keterkecil
print(len(myList)) # Menampilkan jumlah total isi array

# prints out 1,2,3
for x in mylist:
    print(x)
