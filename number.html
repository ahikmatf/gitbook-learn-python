Operator

Operator

Basic operator bisa berupa operasi aritmatika biasa seperti:

  • Pertambahan: c = a + b
  • Pengurangan: c = a - b
  • Pembagian: c = a / b
  • Perkalian: c = a * b

Salah satu kelebihan dari python, ketika memberi value kedalam variabel tidak perlu untuk mendekalarasiakan type datanya.

Contoh operasi aritmatika

#operasi aritmatika dasar

a = 8
b = 10
c = a + b
print(c) #output 18

#operasi aritmatika dasar

number = 1 + 2 * 3 / 4.0
print(number) #output 2.5

#operasi modulus atau sisa pembagian

remainder = 11 % 3
print(remainder) #output 2

#operasi bilangan kuadrat

squared = 7 ** 2 #sama dengan 7*7
cubed = 2 ** 3 #sama dengan 2*2*2
print(squared) #output 49
print(cubed) #output 8

str1 = 'Saya belajar '
print(str1) #output Saya belajar

str2 = "bahasa pemrograman python"
print(str1 + str2) #output Saya belajar bahasa python

bil = 3
print("Saya anak ke -"+bil) #output error TypeError: can only concatenate str (not "int") to str

bil = 3
bil2 = 5
print("Saya anak ke -%d"%bil) #output Saya anak ke -3
print("Saya anak ke -%d dari %d bersaudara"%(bil, bil2)) #output Saya anak ke -3 dari 5 bersaudara

const = 3.15
jari2 = 9
namaBangunRuang = "Lingkaran"
luasLingkaran = const * 9**2
print("Luas dari %s dengan panjang jari-jari %dcm serta dengan konstanta %d adalah %fcm2"%(namaBangunRuang, jari2, const, luasLingkaran))

hellos = "hello " * 10
print(hellos) #output hello hello hello hello hello hello hello hello hello hello

genap = [2,4,6,8]
ganjil = [1,3,5,7]
array = genap + ganjil
print(array)

print([1,2,3] * 3) #output [1, 2, 3, 1, 2, 3, 1, 2, 3]
