Kondisi

Dalam python untuk membandingkan dua nilai dalam suatu kondisi untuk menentukan kondisi tersebut benar atau salah dapat menggunakan operator sama dengan "==", sedangkan tidak sama dengan dapat menggunakan operator "!=".

x = 3
y = 10
print(x == 3) #output True
print(x <= 3) #output True
print(x < 3) #output False
print(x > 3) #output False
print(x < y) #output True

print(True and not False and False) #output False
print(True or False or False) #output True
